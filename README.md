Voice Keyboard Switcher v1.1
Что делает программа - переключатель раскладки и распознавание речи в текст (оффлайн).
По умолчанию переключение - правый shift - RU, левый шифт - EN,
Пока нажат Caps Lock - идёт распознавание голоса в текст, при отпускании текст вставляется в позицию курсора.
Поддерживается распознавание русской и английской речи.
Горячие клавиши могут меняться в настройках (открываются в контекстном меню по правой кнопке мыши).
Скомпилирована для работы в Windows 7/10/11 и т.п.
Донаты на развитие программы принимаются на USDT (BSC BEP-20) 0x4Dc78C4E83C3C48a450A10d9f1B984e5EEEa28B0
О проблемах в работе можете писать в Issues.
